# Ansible-Lab

This is my ansible read-me file. <br><br>

This repository was created to demonstrate competence in automating with Ansible.<br><br>

I installed two RHEL VM's called Bridger and Lexi.<br>
Created a SSH key on local machine (Bridger).<br>
Copied public SSH key to target VM (Lexi).<br>
Created script with Nano.<br>
Used YAML file to define an Ansible Playbook.<br>
Used playbook to install a nginx package to both VM's at the same time.<br>
