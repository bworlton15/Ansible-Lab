# Ansible-Lab

This is my ansible read-me file. 

This repository was created to demonstrate competence in automating with Ansible. 

I installed two RHEL VM's called Bridger and Lexi.
Created a SSH key on local machine (Bridger).
Copied public SSH key to target VM (Lexi).
Created script with Nano.
Used YAML file to define an Ansible Playbook.
Used playbook to install a nginx package to both VM's at the same time.
